서비스 소개

Munny는 단순히 지출을 기록하는 가계부를 넘어, 사용자가 자신의 재정에 대해 자연스럽게 묻고 답할 수 있는 AI 기반 개인 자산 관리 서비스입니다. 이번 달 지출 패턴부터 다음 달 예산 계획까지, 사용자의 데이터를 기반으로 깊이 있는 인사이트를 제공합니다. 한국 사용자의 소비·자산·저축 흐름에 최적화된 분석을 통해, 가계부와 진정한 대화가 가능한 새로운 경험을 만들어갑니다.

주요 기능

💬

대화형 분석

"이번 달 왜 더 썼어?" 같은 질문에 AI가 데이터로 직접 답합니다.

📊

2단계 인사이트

가벼운 즉답부터 심층 분석까지, 깊이를 사용자가 선택합니다.

🏠

자산 통합 관리

지출·자산·저축·부채를 한곳에서 흐름으로 봅니다.

🔒

안전한 데이터

사용자 데이터는 서비스 제공 목적으로만 사용되며 안전하게 보호됩니다.
